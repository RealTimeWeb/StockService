A structured representation of stock information, including ticker symbol, latest sale price, and price change since yesterday.

  • id — The unique ID number for this ticker symbol

  • ticker — The Ticker Symbol (e.g. AAPL)

  • exchange — The name of the exchange (e.g. NASDAQ)

  • last — The latest sale price for this stock.

  • last-trade-date — The entire date of the last trade.

  • last-trade-time — The time of the last trade.

  • change — The price change since yesterday.

  • percent-change — The percent price change since yesterday.
